# Personal Habit Tracking System

A full-stack habit tracking web application built with React, Vite, Java Spring Boot, Spring Security, JWT, and MongoDB.

## Features

- User authentication with JWT
- Habit management
- Daily routine planner
- Calendar and progress tracking
- Journals and mood notes
- Stats dashboard and badges
- Profile management

## Structure

- `backend/` — Spring Boot application
- `frontend/` — React + Vite application

## Setup

### Backend
1. Open `backend/`
2. Configure MongoDB URI in `src/main/resources/application.properties`
3. Run with Maven

### Frontend
1. Open `frontend/`
2. Run `npm install`
3. Run `npm run dev`
